# api-students
[![serverless](http://public.serverless.com/badges/v3.svg)](http://www.serverless.com)

API that manages all students related information. 


### Environment Requirements

- JDK 8
- NodeJS and npm
- Docker (optional)

&nbsp;
### SetUp

    make init

&nbsp;
### Build

    make build

&nbsp;    
### Deploy

    make deploy
